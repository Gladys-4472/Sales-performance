# Sales-performance
Sales performances analysis using Microsoft excel ( pivot Tables, Dashboard, Business) 
# Question answered by the anlysis 
- what is the overall sales performance
- which region generate the most sales
- which product categtories performs best
- what are the monthly sales trends
- which product drives the most profit
- what is the average order value
- How can the business improve sales performance  

# Project Overview
This project analyzes a sales dateset to understand business performance, revenue trends and key drivers os sales using microsoft Ezcel.
The goal is to transform raw sales data into meaningful insight using data cleaning pivot tables and interactive dashboard.
# Tools used 
- Microsoft Excel
- pivot Tables
- Pivot Charts
- Data cleaning
- Business KPI Analysis
# Analysis Performed 
### Data Cleaning  
- Removed duplicates
- standardized colums formats
- Checked missing Values
- Created calculated colums
### Pivot Table analysis
Several Pivot table were created to analysis 
- Total Sales by region
- Sales by product Category
- profit by product
- Monthly sales Trends
### KPI metrics 
key performance indicators calculated 
- Total sales
- total profit
- Average order Value
- Total Quality Sold
